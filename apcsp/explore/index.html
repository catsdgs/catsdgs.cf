Response 2D

Most 3D printers only require the printing file (usually in gcode). This is really all the data that the printer uses. The gcode is then used to position the motors for the print. This is done many times (usually repeated) to make the final print look smoother. Some printers cure resin to print, making the print smoother, which uses the same code, but is processed differently. The print file also contains information on when to extrude the fillament.

If the file is sent over the internet, like some printers, some of the information might be lost. if even one little detail in the code is lost, the whole print could be ruined. Although this is rare, it is a concern in some commercial cases where small details matter.
