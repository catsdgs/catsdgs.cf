Response 2A

The innovation that I have chosen is a 3D Printer. A 3D printer is a printer that can print in 3 dimensions using filament. This filament is usually a type of plastic: PLA, ABS, and others. SLA, DLP, SLS, and EBM printers are the most common types of 3D printers that you will find making small objects. Most 3D printers can print basically anything that you design using 3D design software on your computer, but some industrial machines are programmed to print specific objects. 3D printers are used for many different tasks, from printing small trinkets to printing machine casings. Response 2D

Most 3D printers only require the printing file, usually compiled into one of the printing formats by the slicing software (the software that slices the model into printable layers). One of the more common formats is STL (Standard Tessellation Language). Some of the other formats include OBJ (Object), AMF (Advanced Module File), and 3ML. The code is then used to position the motors for the print. This is done many times (usually repeated) to make the final print look smoother. Some printers cure resin to print, making the print smoother, which uses the same code, but is processed differently. The print file also contains information on when to extrude the filament. If the file is sent over the Internet, like some printers, some of the information might be lost. if even one little detail in the code is lost, the whole print could be ruined. Although this is rare, it is a concern in some commercial cases where small details matter. 3D printers usually have one motor for each axis: X, Y, and Z. These motors spin back and forth to position the head. The head has an extruder which heats up the end of a spool of filament to the perfect temperature where the filament dries soon after it is extruded. This temperature varies depending on the type of filament. A motor and wheel extrude the filament on to the platform. Some 3D printers have a heated platform to help assist the object removal process.
